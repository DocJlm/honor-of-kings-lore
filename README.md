## 🌐 在线阅读 · Live Site

**主入口（GitHub Pages）**：<https://winbeau.github.io/honor-of-kings-lore/>

> ⚠️ **第一次访问 404？** 仓库需要先启用 GitHub Pages：
> 1. 把代码推送到 `main` 分支 → 触发自动构建
> 2. 在 GitHub 仓库页 **Settings → Pages → Source** 选 **GitHub Actions**
> 3. 等待约 1 分钟，Actions 会自动部署到上面的 URL
>
> 之后每次 push 到 `main`，站点自动重新发布。把上述链接中的 `winbeau` 换成你的 GitHub 用户名即可。

## 💻 本地运行 · Run Locally

```bash
git clone https://github.com/winbeau/honor-of-kings-lore
cd honor-of-kings-lore
npm install
npm run dev         # → http://localhost:5173/
```

构建静态站：

```bash
npm run build       # 输出到 docs/.vitepress/dist
npm run preview     # 本地预览构建产物
```
